Methodology, Fan Families, and References

How the tool solves the problem

The tool scales each candidate to the selected speed and density, constructs the active system curve, solves all fan/system intersections, then selects the stable operating point for reporting. Duty margin is checked at the target flow, not at the operating point, because that is the actual selection question.

How to compare architectures correctly

  • Axial fans favor high flow and low pressure.
  • Backward-curved centrifugal fans favor efficiency and non-overloading power behavior.
  • Forward-curved centrifugal fans can create dual-intersection instability in the hump region.
  • Mixed-flow fans sit between axial and centrifugal behavior and are useful when footprint and pressure both matter.

Pressure basis and efficiency

The tool does not convert static and total pressure automatically because that requires velocity information. A basis mismatch is therefore treated as a blocking validity error. Efficiency is computed using the candidate's own basis only when power data is available and the comparison basis matches.

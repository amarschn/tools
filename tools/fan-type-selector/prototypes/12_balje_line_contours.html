Prototype L: Balje Line Contours

A direct Balje-style read: efficiency contours across the full N_s-D_s field, with the optimum Cordier path overlaid as a single dark locus.

Look classic line sheet
Core idea efficiency field first

This is the closest to the textbook distinction: Cordier is one line inside a broader Balje map. Same coordinate system, different amount of information.

What It Emphasizes

The contour field shows how efficiency changes off the optimum path, rather than only where the optimum path itself sits.

Why It Helps

If the user asks “how bad is it if I move off the line?”, this chart can answer that visually.

Prototype C — Digitized Data Points + Monotone Cubic Interpolation

Instead of a formula, this uses reference points digitized from Balje (1981) with smooth interpolation. Efficiency islands are drawn as filled gradient bands.

Approach: 12 reference points were read from Balje's published Cordier chart. Between points, a monotone cubic Hermite spline (Fritsch-Carlson) is used in ln-ln space so the curve is smooth and preserves the shape of the original data without overshooting.

Trade-off vs. a formula: A spline matches the original diagram more faithfully but can't be written as a one-line equation. If adopted, the solver would need a lookup+interpolation function instead of the current closed-form expression.

Prototype D: Verdict Then Evidence

The decision is the first sentence on the page, not the last. A numbered funnel shows how four types narrowed to one, with each step answerable in one click. No nested tabs, no orphaned metric boxes.

Pattern Decision funnel with inline expansion
Goal Make the "why" impossible to miss
How the tool got there
Takeaway

This layout inverts the current structure: verdict first, evidence in a linear funnel, numbers only where they support a specific reasoning step. The user never has to wonder "where is the actual recommendation?" because it's the opening statement.

Prototype R: Balje Atlas (Hero)

A faithful Balje-atlas rebuild. Each machine family (drag, centrifugal, mixed-flow, axial) is a genuine efficiency island — an independent Gaussian peak in log-log (Ns, Ds) — not a slice through one shared ridge. That is what makes the published figure read as a map of territories rather than a curve with banding.

Reference Concepts NREC, Balje (1981)
Duty 1.00 m³/s at 500 Pa
Lead BC Centrifugal

How to read it

Each shaded blob is a region where that family's potential peak efficiency stays above the labelled contour. The heavy black curve is the Cordier optimum locus — it threads the ridge of each island.

Duty point

The white diamond is the current duty-point screen for BC Centrifugal at Ns = 0.80, Ds = 2.34.

Model

η(Ns, Ds) = maxi ηi · exp(−qi), where qi is a rotated bivariate Gaussian centred on the family's optimum point, with negative correlation so each island tilts along the Cordier line.

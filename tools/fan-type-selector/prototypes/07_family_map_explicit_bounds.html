Prototype G: Explicit Bounds Bands

The chart stops pretending the family territories are naturally freehand. Each family is drawn from declared comfort bands plus a softer fade band, so the user can read what defines the bounds without guessing from an irregular outline.

Pattern Comfort band plus fade band
Goal Make the family bounds explicit
Family Map Concept

Bounds Are Data, Not Artwork

The visible field is derived from the union of all family fade bands plus a small pad. The colored core is the comfort zone; the lighter shell is the softer fade zone.

Viewport Basis

Selected Family

Why This Direction Helps
  • The user can see both the declared comfort range and the softer heuristic fade range.
  • The axis bounds are explained by one visible rule instead of hidden `xMax` and `yMax` constants.
  • The map can later be generated from a single structured family dataset instead of hand-tuned polygon points.

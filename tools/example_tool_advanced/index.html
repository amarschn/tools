About This Template

This is the Advanced Example Tool template demonstrating all UI patterns for complex engineering tools:

  • Loading overlay during Pyodide initialization
  • Database-driven dropdowns for material selection
  • Advanced inputs toggle for progressive simplicity
  • Slider with live value for intuitive parameter adjustment
  • Clickable result cards that expand to show derivations
  • Safety factor gauges with color-coded status
  • Status banner with recommendations
  • Interactive charts with explanations
  • Comprehensive Background tab with TOC

Copy this template when building complex tools. See AGENTS.md and DESIGN.md for full documentation.

    3. Interpreting Results

    Safety Factor Guidelines

    SF ≥ Design Factor: Design is acceptable with appropriate margin.
    1.0 ≤ SF < Design Factor: Design may be marginal. Review assumptions and consider increasing capacity.
    SF < 1.0: Design is unacceptable. Failure is predicted.

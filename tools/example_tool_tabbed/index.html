Tool Purpose & README

Purpose of This Tool

This tool serves as the primary template for all other tools in the project. It is fully functional and demonstrates:

  • Loading multiple Python modules from the /pycalcs/ directory.
  • Using utils.py to parse the docstring of example.py.
  • Dynamically populating input tooltips from the docstring.
  • Dynamically building the "Results" section based on the docstring's "Returns" block.

To create a new tool, copy this HTML file and the example.py module, then update the JavaScript constants to point to your new files.

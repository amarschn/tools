1. Air Properties Tfilm = 33.6 °C

Film temperature
Tfilm = (Tb + T∞) / 2 = (42.2 + 25.0) / 2 = 33.6 °C = 306.8 K
Evaluated properties (Sutherland-law fits, valid 200–700 K)
ρ = 1.151 kg/m³  ·  μ = 1.878 × 10⁻⁵ Pa·s  ·  kair = 0.02678 W/m·K
cp = 1007 J/kg·K  ·  Pr = 0.705  ·  ν = 1.632 × 10⁻⁵ m²/s

2. Flow Regime Re = 1213 (laminar)

Channel velocity
Aopen = Nch · s · H = 14 × 0.00464 × 0.030 = 0.001950 m²
Vch = V̇ / Aopen = 0.004800 / 0.001950 = 2.462 m/s
Reynolds number
Re = ρ · Vch · Dh / μ = 1.151 × 2.462 × 0.00804 / 1.878 × 10⁻⁵ = 1213
Re < 2300 → laminar flow regime

3. Convection Coefficient hconv = 31.67 W/m²K

Nusselt number — Muzychka/Yovanovich (2004)
Gz = Re · Pr · Dh/L = 1213 × 0.705 × 0.00804 / 0.100 = 68.8
Nu = (7.54³ + (1.841 · Gz1/3)³)1/3 = (428.5 + 427.9)1/3 = 9.502
Convection coefficient
hconv = Nu · kair / Dh = 9.502 × 0.02678 / 0.00804 = 31.67 W/m²K

Muzychka, Yovanovich, J. Heat Transfer 126(1), 2004. Valid for developing laminar flow in rectangular ducts.

4. Radiation Coefficient hrad = 0.746 W/m²K

Linearized Stefan-Boltzmann
hrad = ε · σ · (Ts² + T∞²)(Ts + T∞) · Feff
= 0.85 × 5.670e-8 × (315.4² + 298.2²)(315.4 + 298.2) × 0.134
= 0.746 W/m²K
View factor
Feff = s / (s + H) = 0.00464 / (0.00464 + 0.030) = 0.134

5. Fin & Surface Efficiency ηf = 90.9%, ηo = 91.7%

Combined heat transfer coefficient
heff = hconv + hrad = 31.67 + 0.746 = 32.42 W/m²K
Fin parameter
m = √(heff · P / (k · Ac)) = √(32.42 × 0.202 / (200 × 0.0001)) = 18.10 m⁻¹
Lc = H + t/2 = 0.030 + 0.0005 = 0.0305 m
m · Lc = 18.10 × 0.0305 = 0.5519
ηf = tanh(0.5519) / 0.5519 = 0.9095
ηo = 1 − (Af/At)(1−ηf) = 1 − (0.090/0.098)(1−0.910) = 0.9167

6. Base Temperature Tb = 42.2 °C

Energy balance solved by bisection
Qrejected = ηo · heff · At · (Tb − T∞)
= 0.9167 × 32.42 × 0.09800 × (42.2 − 25.0) = 50.0 W ✓
Rθ,sink = (Tb − T∞) / Q = 17.19 / 50.0 = 0.3438 K/W

7. Pressure Drop ΔP = 1.33 Pa

Friction factor — Shah & London (1978)
β = s/H = 0.00464/0.030 = 0.155
f · Re = 24(1 − 1.355β + 1.947β² − ...) = 19.93
f = 19.93 / 1213 = 0.01644
Minor losses — Kays & London (1984)
σ = Aopen/Afrontal = 0.813
Kc = 0.42(1 − σ²) = 0.1427
Ke = (1 − σ)² = 0.0352
ΔP = (f·L/Dh + Kc + Ke) · ρV²/2
= (0.01644 × 0.100/0.00804 + 0.143 + 0.035) × 1.151 × 2.46² / 2 = 1.33 Pa

8. Spreading Resistance Rsp = 0.1396 K/W

Yovanovich dimensionless model
As = 20 × 20 mm = 4.0 cm²  ·  Ap = 100 × 80 mm = 80 cm²
ε = √(As/Ap) = √(4.0/80) = 0.2236
τ = t / √Ap = 5.0 / √80 = 0.559
ψ = (1−ε)1.5 / (ε·τ + (1−ε)1.5) = 0.9897
Rsp = ψ / (k · √(π · As)) = 0.9897 / (200 × 0.03544) = 0.1396 K/W

Yovanovich, Muzychka, Culham (1999). Centered rectangular source, uniform flux, adiabatic edges.

9. Interface Resistance Rθ,cs = 0.5000 K/W

TIM mode: conductivity
kTIM = 3.0 W/m·K  ·  BLT = 0.50 mm  ·  A = source footprint = 4.0 cm²
Rθ,cs = BLT / (k · A) = 0.0005 / (3.0 × 0.0004) = 0.4167 K/W

10. Junction Temperature Tj = 89.1 °C

Effective base temperature (includes spreading)
Tbase,eff = Tb + Q · Rsp = 42.2 + 50.0 × 0.1396 = 49.2 °C
Case temperature
Tcase = Tbase,eff + Q · Rθ,cs = 49.2 + 50.0 × 0.5000 = 74.2 °C
Junction temperature
Tj = Tcase + Q · Rθ,jc = 74.2 + 50.0 × 0.3000 = 89.1 °C

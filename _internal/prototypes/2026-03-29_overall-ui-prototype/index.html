Principles

Prototype Focus

  • Make revision, build date, and tool status visible without cluttering the primary task flow.
  • Use report-grade figure framing: axes with units, legend always present, restrained annotation, watermark on exported screenshots.
  • Keep the default layout minimal and practical, then expose more review detail only when expert mode is enabled.
  • Bias toward industrial clarity rather than consumer-app styling.
